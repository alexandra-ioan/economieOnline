Oferta

Oferta = cantitatea dintr-un anumit bun ce poate fi vanduta pe piata in functie de anumiti factori.

Evolutia ofertei in functie de pret este evidentiata de legea ofertei:

Elasticitatea ofertei in functie de pret exprima intensitatea modificarii cantitatii in functie de intensitatea modificarii pretului.

Elasticitatea ofertei in functie de pret:

Kec/p =
Δ%Qo / Δ%P
=
ΔQo / Qo0
:
ΔP / Po0


  • Oferta elastica: |Δ%Qo| > |Δ%P| => Keo/p > 1

  • Oferta de elasticitate unitara: Δ%Qo = Δ%P => Keo/p = 1

  • Oferta inelastica: |Δ%Qo| < |Δ%P| => Keo/p < 1
